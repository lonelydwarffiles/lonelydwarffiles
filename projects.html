Let There be Light! - Using a Raspberry Pi

Project 1 - Computer Science

This project is to turn on a light using a Raspberry Pi 2, it uses some code displayed below. I worked on this project with a friend named Will.

My plan for success was to write code in order to get the LED to turn on when a button was clicked. So I used python to code a thing so that the output is turned on when you run the code. My plan was then to have it so that when the program is running, a command could be inputed into the shell and using the arguments that the command provided it would either turn on the light or turn on via the input provided.

My plan worked up to a point. I mean this by the fact that I was not able to get the keyboard commands to work. The progam to turn on the LED did work pretty well until one of the LEDs blew.
